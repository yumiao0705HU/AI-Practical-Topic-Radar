# Scoring Rubric

Score each candidate from 1 to 5.

## 5 - Strong Recommend

- Clear practical use case
- Reproducible in 30-90 minutes
- Evidence link is available
- China landing path is realistic
- Output artifact is obvious
- Low or manageable risk

## 4 - Recommend

- Useful and timely
- Some setup or cost friction
- Evidence is acceptable
- Practical path is clear with minor assumptions

## 3 - Watch

- Interesting but not yet easy to reproduce
- Demand signal exists but evidence is thin
- China landing path needs more checking
- Better for tracking than immediate tutorial/demo

## 2 - Weak

- Mostly hype or marketing
- Hard to reproduce
- Weak source evidence
- Tool access or setup is too uncertain

## 1 - Skip

- No practical path
- Source cannot be verified
- Stale, broken, misleading, unsafe, or not China-landable

## Tie-breakers

Prefer candidates that:

- Help users make something visible
- Solve a repeated pain point
- Can become a template, SOP, comparison, or demo
- Have Chinese-friendly access or alternatives
- Are useful beyond a single platform trend

Downgrade candidates that:

- Require expensive paid plans for basic testing
- Depend on unstable unofficial APIs
- Require hard-to-access overseas services with no workaround
- Are likely to mislead beginners
- Need sensitive credentials or unsafe local permissions
