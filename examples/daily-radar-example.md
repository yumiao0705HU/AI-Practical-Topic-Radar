# Example Daily Radar Output

今日主题：AI 实操机会集中在具体案例：知识库、日记、视频、图片、网站和自动化工作流。

1. **AI Agent + 个人知识库问答**
   - 案例类型：知识库
   - 来源：B站 / GitHub / 官方文档
   - 适合谁：学生 / 研究者 / 企业团队
   - 这个案例具体做什么：把本地文档或笔记接入知识库，让 Agent 回答问题并给出处。
   - 为什么值得看：这是最容易被普通用户理解和复现的 Agent 案例之一。
   - 中国可落地性：优先选择可本地部署或支持国产模型的方案。
   - 实操路径：90 分钟内整理 10 篇文档，搭一个最小知识库问答 demo。
   - 可交付物：demo + 搭建步骤 + 踩坑清单
   - 证据：教程或仓库链接，访问于 YYYY-MM-DD
   - 难度：中
   - 风险：文档切分、召回质量和模型成本需要调试。
   - 推荐动作：做 demo

2. **AI Agent + 日记复盘助手**
   - 案例类型：日记
   - 来源：小红书 / 网页搜索
   - 适合谁：普通 AI 用户 / 创作者 / 独立开发者
   - 这个案例具体做什么：把每天的碎片记录整理成复盘、情绪趋势和下一步计划。
   - 为什么值得看：比抽象效率工具更贴近日常使用。
   - 中国可落地性：可用飞书、多维表格、Notion 替代、Obsidian 或本地 Markdown。
   - 实操路径：30 分钟内准备 7 条日记样本，生成一份周复盘。
   - 可交付物：日记模板 + 复盘 prompt + 示例输出
   - 证据：帖子或教程链接，访问于 YYYY-MM-DD
   - 难度：低
   - 风险：隐私和过度解读。
   - 推荐动作：做教程
